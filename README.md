# BEV.CUTS Website

Premium mobile-first barber website built with Next.js 15, TypeScript, Tailwind CSS, Framer Motion, and Embla Carousel.

## Development

```bash
npm install
npm run dev
```

## Gallery Videos

Drop `.mp4` transformation videos into `public/gallery`, then update the single array in `lib/gallery.ts`.

Booking and social links live in `lib/site.ts`.
